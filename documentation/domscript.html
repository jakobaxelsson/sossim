domscript

This modules generates basic functionality for creating DOM trees in a Pythonic fashion. It is inspired by the Python dominate library in its use of context managers to describe the tree structure. However, it builds the tree directly, instead of creating a temporary structure from which HTML text is generated. This also makes it possible to add event listeners, modify the DOM dynamically, etc.

A typical usage looks as follows:

from domscript import *

Get the body element of the document, clearing its current children, and adding new ones.

with document.query(".body").clear(): with ol(): with ul(): with li("Item 1.1"): event_listener("click", lambda _: print("Click")) li("Item 1.2") with li("Item 2") as item: item["id"] = "item2" li("Item 3", id = "item3")

 65class DomWrapper:
 66    """
 67    A class that acts as a context manager for DOM element creation functions.
 68    """
 69
 70    # Keep av stack of surrounding contexts.
 71    stack: ClassVar[List[Self]] = []

 73    def __init__(self, tag_name: str, content: Optional[str | Self] = None, namespace: Optional[str] = None, **attrs: Any):
 74        """
 75        Create the new DOM node with the given tag_name.
 76        If content is provided as a string, it is assigned to the node as inner HTML.
 77        If content is provided as a tag, it is added as a child.
 78
 79        Args:
 80            tag_name (str): the tag name.
 81            content (str | Self, optional): content of the DOM node. Defaults to None.
 82            namespace (str, optional): a name space string. Defaults to None.
 83            attrs (Any): a dictionary of attribute values. 
 84        """
 85        if isinstance(namespace, str):
 86            self.dom_element = js.document.createElementNS(namespace, tag_name)
 87        else:
 88            self.dom_element = js.document.createElement(tag_name)
 89
 90        # If some content was provided, add it to the node depending on its type.
 91        if isinstance(content, str):
 92            # If it is a string, add it as inner HTML
 93            self.dom_element.innerHTML = content
 94        elif isinstance(content, DomWrapper):
 95            # Otherwise, assume it is a DOM node, and add it as a child
 96            self.dom_element.appendChild(content.dom_element)
 97
 98        # If attributes were provided, add them to the node, mapping the names to avoid clashes with Python reserved words.
 99        for (a, v) in attrs.items():
100            self.dom_element.setAttribute(self.map_attribute_name(a), v)
101
102        # If this element is created inside a context, then add it as a child of its parent.
103        if DomWrapper.stack != []:
104            DomWrapper.stack[-1].dom_element.appendChild(self.dom_element)
105
106    def __enter__(self) -> Self:
107        """
108        Enter context. Push this element onto the stack, so that it becomes the parent of elements created within the context.
109
110        Returns:
111            Self: returns self.
112        """
113        DomWrapper.stack.append(self)
114        # Return the created DOM element so that it can be bound to the context variable.
115        return self
116
117    def __exit__(self, exc_type, exc_value, traceback):
118        """
119        Pops the top element of the stack to return to the outer context.
120        """
121        DomWrapper.stack.pop()

Base class for protocol classes.

Protocol classes are defined as::

class Proto(Protocol):
    def meth(self) -> int:
        ...

Such classes are primarily used with static type checkers that recognize structural subtyping (static duck-typing), for example::

class C:
    def meth(self) -> int:
        return 0

    return x.meth()
